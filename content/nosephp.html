Welcome to NosePHP

This default NosePHP website acts as a demo for the main modules available.

HTML module

The content on this page was loaded from a HTML file via NosePHPs html module, a powerful module for adding static content.

Markdown module

The content on the Markdown page is loaded and parsed from a Markdown file using NosePHPs markdown module, another powerful module for adding static pages.

Text module

The content on the Text page is the same Markdown file loaded using NosePHPs text module, which is useful if you want a site to display text file content unformatted.

WordPress module

The content on the WordPress page is dynamic content loaded from a WordPress blog via the WordPress API. This powerful module will work with any WordPress blog, including free blogs on WordPress.com.
